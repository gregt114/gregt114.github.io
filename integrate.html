Numerical Integration








Result:

This page can use various integration algorithms to compute

$$ \int_a^b f(x)dx $$

using \(\ n \ \) sampled points.

Note that \(\ f(x)\ \) must be specifed using valid javascript syntax. For example, use the following substitutions:

\(\sin(x) \rightarrow \ \) Math.sin(x)

\(\pi \rightarrow \ \) Math.PI

\( e \rightarrow \ \) Math.E

\(\sqrt{x} \rightarrow \ \) Math.sqrt(x)

\(\ \ln(x) \rightarrow \ \) Math.log(x)

* for multiplication and ** for exponentiation
